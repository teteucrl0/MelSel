# Ícones animados (Lottie)

Usados **somente** onde o significado bate com a tela:

| Arquivo | Conteúdo | Uso |
|---------|----------|-----|
| bee.json | Inseto voador (LottieFiles) | Loading de busca / catálogo |
| search.json | Lupa | (reservado) |
| package.json | Pin de local | Pedidos / entrega |

Arquivos antigos com animações genéricas (atendimento, engrenagem, moedas) **não** são mais usados no app.

**Apicultor / fornecedor no cadastro:** ícone estático de abelha (`flaticon/bee.png`), não animação Lordicon incorreta.

**Atribuição:** [Lordicon](https://lordicon.com/) / [LottieFiles](https://lottiefiles.com/) conforme origem de cada JSON.

Ícones estáticos: `../flaticon/ATTRIBUTION.md`
